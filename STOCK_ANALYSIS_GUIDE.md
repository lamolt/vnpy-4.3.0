# VeighNa 股票分析使用指南

## 一、软件概述

VeighNa（vnpy）是一套开源量化交易框架，支持股票、期货、期权等多种金融产品的分析交易。本指南介绍如何使用其核心功能进行股票技术分析。

## 二、安装与启动

### 1. 启动VeighNa Trader
```bash
python run_vnpy.py
```

### 2. 连接数据服务
- 点击 **系统** → **数据服务** → 选择 **迅投研** 或 **RQData**
- 输入账号密码完成连接

## 三、股票行情获取

### 1. 连接券商接口
- 点击 **系统** → **连接CTP**（使用SimNow仿真账号测试）
- 或连接其他股票交易接口（中泰XTP、华鑫奇点等）

### 2. 订阅行情
在交易接口连接成功后，系统会自动订阅已添加的合约行情。

## 四、K线图表分析

VeighNa提供强大的K线图表功能：

1. **打开图表**：点击 **功能** → **K线图表**
2. **选择标的**：在界面中选择要分析的股票代码
3. **切换周期**：支持1分钟、5分钟、15分钟、30分钟、60分钟、日线、周线等
4. **添加技术指标**：
   - MA（均线）：右键图表 → 添加指标 → 选择MA
   - MACD、KDJ、布林带等常用指标均可添加

## 五、策略回测

### CTA策略回测
1. 点击 **功能** → **CTA回测**
2. 选择策略模板（如均值回归策略、布林带策略）
3. 设置回测参数：
   - 合约代码
   - 时间范围
   - 初始资金
   - 手续费率
4. 点击 **开始回测** 查看结果

### 回测结果分析
- **收益率曲线**：查看策略整体表现
- **最大回撤**：评估风险
- **夏普比率**：衡量风险收益比
- **交易明细**：每笔交易的详细记录

## 六、注意事项

1. **数据质量**：确保行情数据完整准确
2. **回测与实盘差异**：实盘交易需考虑滑点、流动性等因素
3. **风险控制**：设置合理的止损止盈规则
4. **仿真测试**：实盘前务必在仿真环境充分测试

## 七、进阶功能

- **算法交易**：使用TWAP、Sniper等算法执行大单
- **组合策略**：同时管理多合约策略
- **风险管理**：实时监控持仓风险

## 八、总结

VeighNa提供从行情获取、技术分析、策略回测到实盘交易的一站式解决方案，适合有一定Python基础的量化交易者使用。建议先从仿真测试开始，逐步验证策略有效性后再投入实盘。
